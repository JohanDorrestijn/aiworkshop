# AI Workshop - OpenEdge ABL Project

This project demonstrates OpenEdge ABL development patterns and best practices for building business applications.

## Project Structure

- `src/business/` - Business logic classes and datasets
- `src/` - User interface windows
- `doc/` - Documentation and patterns
- `dump/` - Database schema files
- `.windsurf/` - Development workflows and rules

## Key Features

- Business Entity Pattern implementation
- Customer management system
- OpenEdge UI windows
- Configuration management

## Getting Started

This project requires OpenEdge ABL development environment and the sports2000 database schema.
